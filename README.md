# 东软4S教育平台自动化工具

本项目是一个自动化工具，用于处理东软4S教育平台的相关操作，包括课程访问、视频观看记录等功能。

## 功能特性

- 自动登录系统
- 验证码识别
- 课程信息获取
- 视频观看记录模拟
- 文档阅读记录模拟

## 安装指南

### 1. 克隆仓库
```bash
git clone https://github.com/yourusername/pachong_for_neu.git
cd pachong_for_neu

### 2. 安装依赖
pip install -r requirements.txt

### 3. 识别验证码模型
确保 models 目录中包含以下文件：
    - 101file_0.875_222_2000_2024-10-01-23-52-44.onnx
    - charsets.json
本项目使用ddddocr开源项目训练识别验证码模型，详细内容请参考原开源ddddocr项目，
url：https://github.com/sml2h3/ddddocr。

### 4. 程序内容解释
user_info.json: 存储用户信息。

### 5. 运行程序
python main.py
随后根据提示输入用户名和密码。

### 6. 注意事项
程序运行期间请保持网络稳定。

## 已知问题与贡献

### 当前版本可能存在的问题

1. **验证码识别稳定性**
   - 验证码识别模型在特定条件下可能识别率下降
   - 复杂验证码可能出现识别错误

2. **网络请求稳定性**
   - 在高延迟网络环境下可能出现请求超时

3. **多线程支持**
   - 当前版本尚未实现多课程并行处理

4. **错误处理**
   - 部分异常情况处理不够完善
   - 错误提示信息可以更加友好
   - 程序在刷课的时候可能会出现时间够了但是任务点还是没显示成功的情况，现在有两种解决方法
      - 第一种：刷完发现任务点还是白色的（没有刷成功），手动点击进入一下这个视频任务点，然后再运行程序就可以了
      - 第二种：刷完发现任务点显示半块蓝色，观看时间是够的但是显示没有完成，此时手动点击进入这个任务就可完成了
   - 程序访问频次过高（不改动代码关键部分就不会）导致账号临时被封，一般情况等一个小时后使用web端手动登陆就可以解除， 是在不行的话联系你们辅导员吧
